# 🤖 AI-Powered Data Analyst Assistant

An advanced, interactive web application built with **Streamlit** and **Groq Cloud (Llama 3.3)** that allows you to analyze, clean, and visualize your data using natural language. No more writing complex Python or SQL queries—just ask your data anything!

## 🚀 Features

- **Multi-Format Support**: Upload and analyze `.csv`, `.xlsx`, and `.json` files.
- **Natural Language Querying**: Ask questions like "What is the average salary?" or "List top 5 products" and get instant answers.
- **Interactive Visualizations**: Generate bar charts, scatter plots, and heatmaps automatically using Seaborn and Matplotlib.
- **Smart Data Overview**: Real-time dashboard showing total rows, columns, duplicates, and missing values.
- **Data Cleaning Tools**: One-click buttons to remove duplicates and fill missing values.
- **Chat History**: Remembers previous questions for a seamless conversational experience.

## 🛠️ Tech Stack

- **Frontend**: [Streamlit](https://streamlit.io/)
- **AI Model**: [Groq Cloud](https://groq.com/) (Llama-3.3-70b-versatile)
- **Data Processing**: Pandas
- **Visualization**: Matplotlib & Seaborn
